## Context

* The diff between the main branch and the current checked out branch: !`git diff-base`
* The commits between the main branch and current checked out branch: !`git log-base`
* The name of the current branch: !`git branch-name`

## Task

Summarise the changes in the current branch. First make an objective description of what changed. Then review the code quality.

Focus Areas:

**Strengths**: What's working well
**Weaknesses**: Key issues or gaps
**Improvements**: Top 2-3 actionable recommendations

Keep output brief and strategic. Avoid detailed explanations unless critical.
